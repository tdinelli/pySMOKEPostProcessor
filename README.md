# pySMOKEPostProcessor

Python binder for the OpenSMOKEppPostProcessor developed by Professor [Alberto Cuoci](http://creckmodeling.chem.polimi.it/menu-people/menu-people-faculty/menu-people-alberto-cuoci). You can find the original repository of the original OpenSMOKEPostProcessor at this link (https://github.com/acuoci/OpenSMOKEppPostProcessor).

## External libraries dependencies

- OpenSMOKEpp distibuted under request contact Alberto Cuoci <alberto.cuoci@polimi.it>
- Eigen freely downloadable at https://eigen.tuxfamily.org/index.php?title=Main_Page
- Boost freely downloadable at https://www.boost.org/
